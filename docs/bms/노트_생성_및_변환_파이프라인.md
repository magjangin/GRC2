## 변환 파이프라인 아키텍처

### 전체 흐름도

```
BMS 파일 파싱
    ↓
List<BmsNote> (BMS 노트 리스트)
    ↓
BmsNoteConverter.ConvertBmsNotesToNoteCreateData()
    ├─ 1. 초기화 검증
    ├─ 2. 캐시 초기화 (NoteCreateDataBuilder.ClearCache())
    ├─ 3. 시간 순 정렬
    ├─ 4. 각 BmsNote 처리
    │   ├─ 홀드/페어리 끝 노트 분리
    │   └─ NoteCreateDataBuilder.CreateNoteCreateData()
    │       ├─ NoteConstructorHelper.TryFindConstructor()
    │       │   ├─ 2~6개 파라미터 생성자 시도
    │       │   └─ 실패 시 기본 생성자 사용
    │       ├─ perfectSample 설정
    │       ├─ laneLeftRightID 설정
    │       ├─ subLaneID 설정
    │       ├─ noteTypeID 설정
    │       └─ NoteFieldInitializer 호출
    │           ├─ SetDirectionIndex()
    │           ├─ SetTurnDirection() (페어리용)
    │           ├─ SetNoteSize()
    │           ├─ SetSlideEndFlickDirection()
    │           └─ SetDefaultBooleanFields()
    ├─ 5. 홀드 끝 노트 처리 (HoldNoteProcessor)
    ├─ 6. 페어리 끝 노트 처리 (FairyNoteProcessor)
    ├─ 7. 0초 더미 노트 필터링
    ├─ 8. 마지막 노트 플래그 설정
    └─ 9. Array 변환 및 반환
    ↓
NoteCreateData[] (게임 노트 배열)
```

### ConvertBmsNotesToNoteCreateData 메서드

**목적**: BMS 노트 리스트를 게임 노트 배열로 변환

```csharp
public static Array ConvertBmsNotesToNoteCreateData(List<BmsNote> bmsNotes)
{
    // ✅ 1단계: 타입 안전성 검증
    if (Loaders.GameTypeLoader.NoteCreateDataType == null)
    {
        MelonLogger.Error("[BmsNoteConverter] 초기화되지 않았습니다.");
        return null;
    }

    if (bmsNotes == null)
    {
        MelonLogger.Error("[BmsNoteConverter] bmsNotes가 null입니다.");
        return null;
    }

    try
    {
        // ⚡ 2단계: 캐시 초기화 (성능 최적화)
        Builders.NoteCreateDataBuilder.ClearCache();

        var noteList = new List<object>();
        var holdEndNotes = new List<BmsNote>();
        var fairyEndNotes = new List<BmsNote>();

        // ⚡ 3단계: 시간 순 정렬 (한 번만)
        var sortedBmsNotes = bmsNotes.OrderBy(n => n.Time).ToList();
        
        // 4단계: 각 노트 변환
        foreach (var bmsNote in sortedBmsNotes)
        {
            // 홀드/페어리 끝 노트는 별도 처리
            if (bmsNote.Type == NoteType.HoldEnd)
            {
                holdEndNotes.Add(bmsNote);
                continue;
            }
            if (bmsNote.Type == NoteType.FairyEnd)
            {
                fairyEndNotes.Add(bmsNote);
                continue;
            }

            // 노트 생성
            var noteCreateData = Builders.NoteCreateDataBuilder.CreateNoteCreateData(bmsNote);
            if (noteCreateData != null)
            {
                noteList.Add(noteCreateData);
            }
        }

        // 5단계: 끝 노트 검증
        var missingHoldEnds = CheckMissingEndNotes(
            bmsNotes.Where(n => n.Type == NoteType.Hold),
            holdEndNotes, "홀드", "홀드 시작 노트(02)", "홀드 끝 노트(19)");
        
        if (missingHoldEnds != null)
        {
            return null; // 주입 금지
        }

        // 6단계: 홀드 끝 노트 처리
        HoldNoteProcessor.ProcessHoldEndNotes(noteList, holdEndNotes, bmsNotes, ...) (구현: HoldNoteProcessor.ConnectNodes.cs);

        // 7단계: 페어리 끝 노트 검증 및 처리
        var missingFairyEnds = CheckMissingEndNotes(
            bmsNotes.Where(n => n.Type == NoteType.Fairy),
            fairyEndNotes, "페어리", "페어리 시작 노트(11-18)", "페어리 끝 노트(1A-1B)");
        
        if (missingFairyEnds != null)
        {
            return null; // 주입 금지
        }

        FairyNoteProcessor.ProcessFairyEndNotes(noteList, fairyEndNotes, bmsNotes, ...) (구현: FairyNoteProcessor.ConnectNodes.cs);

        // 8단계: 0초 더미 노트 필터링
        FilterZeroTimeNotes(noteList);

        // 9단계: 마지막 노트 플래그 설정
        SetLastNoteFlag(noteList);

        // 10단계: 배열 변환
        var array = Array.CreateInstance(Loaders.GameTypeLoader.NoteCreateDataType, noteList.Count);
        for (int i = 0; i < noteList.Count; i++)
        {
            array.SetValue(noteList[i], i);
        }

        return array;
    }
    catch (Exception ex)
    {
        Helpers.ErrorLogger.LogException(ex, "[BmsNoteConverter]", "변환 오류");
        return null;
    }
}
```

### CheckMissingEndNotes 메서드

**목적**: 시작 노트에 대응하는 끝 노트가 있는지 검증

```csharp
private static List<BmsNote> CheckMissingEndNotes(
    IEnumerable<BmsNote> startNotes,
    List<BmsNote> endNotes,
    string noteTypeName,
    string startNoteDescription,
    string endNoteDescription)
{
    var missingEnds = new List<BmsNote>();
    const float TIME_TOLERANCE = 0.01f; // 시간 오차 허용 범위
    
    foreach (var startNote in startNotes)
    {
        var expectedEndTime = startNote.Time + startNote.Duration;
        
        // 끝 노트 찾기: 시간, 레인, Left/Right 모두 일치해야 함
        var hasEndNote = endNotes.Any(end => 
            Math.Abs(end.Time - expectedEndTime) < TIME_TOLERANCE &&
            end.Lane == startNote.Lane &&
            end.IsLeft == startNote.IsLeft);
        
        if (!hasEndNote)
        {
            missingEnds.Add(startNote);
        }
    }
    
    if (missingEnds.Count > 0)
    {
        // 상세한 에러 메시지 출력
        MelonLogger.Error("═══════════════════════════════════════════════════════════════");
        MelonLogger.Error($"❌❌❌ BMS 노트 주입 실패: {noteTypeName} 끝 노트가 없습니다! ❌❌❌");
        MelonLogger.Error("═══════════════════════════════════════════════════════════════");
        
        foreach (var startNote in missingEnds)
        {
            MelonLogger.Error($"   {noteTypeName} 시작: Time={startNote.Time:F3}초, " +
                            $"Lane={startNote.Lane}, IsLeft={startNote.IsLeft}, " +
                            $"Duration={startNote.Duration:F3}초");
            MelonLogger.Error($"   예상 끝 시간: {startNote.Time + startNote.Duration:F3}초");
        }
        
        return missingEnds; // 주입 금지
    }
    
    return null; // 모든 끝 노트가 있음
}
```

### SetLastNoteFlag 메서드

**목적**: 가장 마지막 노트에 isLast 플래그 설정

```csharp
private static void SetLastNoteFlag(List<object> noteList)
{
    int maxPerfectSample = int.MinValue;
    object lastNote = null;

    // 모든 노트와 connectNodeDataArray의 끝 노트를 확인
    foreach (var noteObj in noteList)
    {
        // 메인 노트의 perfectSample 확인
        var perfectSample = Helpers.FieldAccessHelper.GetFieldValue(noteObj, 
            Helpers.FieldAccessHelper.FIELD_PERFECT_SAMPLE);
        
        if (perfectSample != null)
        {
            int sample = (int)perfectSample;
            if (sample > maxPerfectSample)
            {
                maxPerfectSample = sample;
                lastNote = noteObj;
            }
        }

        // connectNodeDataArray의 끝 노트도 확인
        var connectNodeArray = Helpers.FieldAccessHelper.GetFieldValue(noteObj, 
            Helpers.FieldAccessHelper.FIELD_CONNECT_NODE_DATA_ARRAY);
        
        if (connectNodeArray != null && connectNodeArray is Array connectArray)
        {
            foreach (var connectNode in connectArray)
            {
                var connectPerfectSample = Helpers.FieldAccessHelper.GetFieldValue(connectNode, 
                    Helpers.FieldAccessHelper.FIELD_PERFECT_SAMPLE);
                
                if (connectPerfectSample != null)
                {
                    int connectSample = (int)connectPerfectSample;
                    if (connectSample > maxPerfectSample)
                    {
                        maxPerfectSample = connectSample;
                        lastNote = connectNode;
                    }
                }
            }
        }
    }

    // 마지막 노트에 isLast 설정
    if (lastNote != null)
    {
        Helpers.FieldAccessHelper.SetFieldValue(lastNote, 
            Helpers.FieldAccessHelper.FIELD_IS_LAST, true);
    }
}

### CreateNoteCreateData 메서드

**목적**: BmsNote를 NoteCreateData 인스턴스로 변환

```csharp
public static object CreateNoteCreateData(BmsNote bmsNote)
{
    try
    {
        // 1. 타입 검증
        if (GameTypeLoader.NoteCreateDataType == null)
        {
            MelonLogger.Error("[NoteCreateDataBuilder] NoteCreateData 타입이 초기화되지 않았습니다.");
            return null;
        }

        // 2. perfectSample 계산
        var perfectSample = (int)(bmsNote.Time * SAMPLE_RATE);
        
        // 3. Enum 값 가져오기
        var noteTypeId = EnumValueHelper.GetNoteTypeId(bmsNote.Type);
        var laneLeftRight = EnumValueHelper.GetEnumValue(
            GameTypeLoader.NoteLaneLeftRightEnum, 
            bmsNote.IsLeft ? EnumValueHelper.GetEnumLeft() : EnumValueHelper.GetEnumRight());
        
        bool isLeftBool = bmsNote.IsLeft;
        
        // 4. 생성자로 인스턴스 생성 시도
        object noteCreateData = NoteConstructorHelper.TryFindConstructor(
            GameTypeLoader.NoteCreateDataType,
            bmsNote,
            perfectSample,
            noteTypeId,
            isLeftBool);
        
        bool usedDefaultConstructor = false;
        
        // 5. 생성자 실패 시 기본 생성자 사용
        if (noteCreateData == null)
        {
            noteCreateData = Activator.CreateInstance(GameTypeLoader.NoteCreateDataType);
            usedDefaultConstructor = true;
        }

        // 6. perfectSample 설정
        SetPerfectSample(noteCreateData, perfectSample, usedDefaultConstructor);

        // 7. 기본 필드 설정
        FieldAccessHelper.SetFieldValue(noteCreateData, 
            FieldAccessHelper.FIELD_LANE_LEFT_RIGHT_ID, laneLeftRight);

        var subLane = EnumValueHelper.GetSubLaneType(bmsNote.Lane);
        FieldAccessHelper.SetFieldValue(noteCreateData, 
            FieldAccessHelper.FIELD_SUB_LANE_ID, subLane);

        FieldAccessHelper.SetFieldValue(noteCreateData, 
            FieldAccessHelper.FIELD_NOTE_TYPE_ID, noteTypeId);

        // 8. NoteFieldInitializer로 추가 필드 설정
        object directionIndexValue = NoteFieldInitializer.SetDirectionIndex(noteCreateData, bmsNote);

        if (bmsNote.Type == NoteType.Fairy || bmsNote.Type == NoteType.FairyEnd)
        {
            NoteFieldInitializer.SetTurnDirection(noteCreateData, directionIndexValue, bmsNote);
        }

        NoteFieldInitializer.SetNoteSize(noteCreateData);
        NoteFieldInitializer.SetSlideEndFlickDirection(noteCreateData);
        NoteFieldInitializer.SetDefaultBooleanFields(noteCreateData);

        // 9. 최종 검증
        ValidatePerfectSample(noteCreateData, perfectSample, bmsNote);

        return noteCreateData;
    }
    catch (Exception ex)
    {
        ErrorLogger.LogException(ex, "[NoteCreateDataBuilder]", "CreateNoteCreateData 오류");
        return null;
    }
}
```

### SetPerfectSample 메서드

**목적**: perfectSample 필드를 안전하게 설정

```csharp
private static void SetPerfectSample(object noteCreateData, int perfectSample, bool usedDefaultConstructor)
{
    var existingPerfectSample = FieldAccessHelper.GetFieldValue(noteCreateData, 
        FieldAccessHelper.FIELD_PERFECT_SAMPLE);
    var existingMSample = FieldAccessHelper.GetFieldValue(noteCreateData, "mSample");
    
    // 기본 생성자를 사용했거나, perfectSample이 null이거나 0인 경우 수동 설정
    bool needsManualSet = usedDefaultConstructor || 
                         (existingPerfectSample == null && existingMSample == null) ||
                         (existingPerfectSample != null && (int)existingPerfectSample == 0) ||
                         (existingMSample != null && (int)existingMSample == 0);
    
    if (needsManualSet)
    {
        // 여러 필드에 설정 (게임 버전에 따라 다를 수 있음)
        FieldAccessHelper.SetFieldValue(noteCreateData, 
            FieldAccessHelper.FIELD_PERFECT_SAMPLE, perfectSample);
        FieldAccessHelper.SetFieldValue(noteCreateData, "mSample", perfectSample);
        FieldAccessHelper.SetFieldValue(noteCreateData, "sample", perfectSample);
    }
    else
    {
        // 값이 다르면 경고
        var actualSample = existingPerfectSample ?? existingMSample;
        if (actualSample != null && !actualSample.Equals(perfectSample))
        {
            MelonLogger.Warning($"[NoteCreateDataBuilder] perfectSample 불일치: " +
                              $"생성자={actualSample}, 예상={perfectSample}");
        }
    }
}
```

### GetBmsNoteFromNoteCreateData 메서드 (캐싱)

**목적**: NoteCreateData에서 원본 BmsNote 찾기 (O(1) 검색)

```csharp
public static BmsNote GetBmsNoteFromNoteCreateData(object noteCreateData, List<BmsNote> bmsNotes)
{
    try
    {
        var perfectSample = FieldAccessHelper.GetFieldValue(noteCreateData, 
            FieldAccessHelper.FIELD_PERFECT_SAMPLE);
        
        if (perfectSample == null)
            return null;

        var perfectSampleInt = (int)perfectSample;
        var time = perfectSampleInt / (float)SAMPLE_RATE;

        // NoteCreateData에서 lane, isLeft, targetType 추출 (laneLeftRightID, subLaneID, noteTypeID)
        bool isLeft = ...; int lane = ...; NoteType? targetType = ...; // typeStr "Fairy"/"Hold"/"Touch"/"Flick"

        // ⚡ 캐시가 없거나 다른 리스트인 경우 새로 생성
        bool needRebuild = _timeToBmsNoteCache == null || 
                           _cachedBmsNotes == null || 
                           !ReferenceEquals(_cachedBmsNotes, bmsNotes);
        
        if (needRebuild)
        {
            _timeToBmsNoteCache = new Dictionary<string, BmsNote>();
            _cachedBmsNotes = bmsNotes;
            // 키: "{timeKey}_{lane}_{isLeft}_{type}" (0.001초 반올림, 동일 시간·레인·타입 구분)
            foreach (var note in bmsNotes)
            {
                var timeKey = (float)Math.Round(note.Time, 3);
                var key = $"{timeKey}_{note.Lane}_{note.IsLeft}_{note.Type}";
                if (!_timeToBmsNoteCache.TryGetValue(key, out _))
                    _timeToBmsNoteCache[key] = note;
            }
        }

        // ⚡ Dictionary에서 O(1) 검색
        var searchTimeKey = (float)Math.Round(time, 3);
        var searchKey = $"{searchTimeKey}_{lane}_{isLeft}_{targetType}";
        if (_timeToBmsNoteCache.TryGetValue(searchKey, out var cachedNote))
        {
            if (Math.Abs(cachedNote.Time - time) < 0.001f)
                return cachedNote;
        }

        // fallback: 리스트 선형 검색
        return bmsNotes.Find(n => Math.Abs(n.Time - time) < 0.001f && n.Lane == lane && n.IsLeft == isLeft && (targetType == null || n.Type == targetType));
    }
    catch
    {
        return null;
    }
}
```

| 방식 | 시간 복잡도 | 1000개 노트 검색 시간 |
|------|-------------|----------------------|
| 선형 검색 (기존) | O(n) | ~50ms |
| Dictionary 캐싱 (최적화) | O(1) | ~0.1ms |

### TryFindConstructor 메서드

**목적**: 여러 생성자 시그니처를 시도하여 인스턴스 생성

```csharp
public static object TryFindConstructor(
    Type noteCreateDataType,
    BmsNote bmsNote,
    int perfectSample,
    object noteTypeId,
    bool isLeftBool)
{
    object noteCreateData = null;
    
    // 생성자 시그니처 정의 (우선순위 순서)
    var constructorSignatures = new[]
    {
        // 2개 파라미터
        new { 
            Types = new Type[] { typeof(float), typeof(int) }, 
            Args = new object[] { bmsNote.Time, bmsNote.Lane }, 
            Desc = "2개 파라미터 (time, lane)" 
        },
        new { 
            Types = new Type[] { typeof(int), typeof(int) }, 
            Args = new object[] { perfectSample, bmsNote.Lane }, 
            Desc = "2개 파라미터 (sample, lane)" 
        },
        
        // 3개 파라미터
        new { 
            Types = new Type[] { typeof(float), typeof(int), noteTypeId.GetType() }, 
            Args = new object[] { bmsNote.Time, bmsNote.Lane, noteTypeId }, 
            Desc = "3개 파라미터 (time, lane, type)" 
        },
        
        // 4개 파라미터
        new { 
            Types = new Type[] { typeof(float), typeof(int), noteTypeId.GetType(), typeof(bool) }, 
            Args = new object[] { bmsNote.Time, bmsNote.Lane, noteTypeId, isLeftBool }, 
            Desc = "4개 파라미터 (time, lane, type, leftRight)" 
        },
        
        // 5개 파라미터
        new { 
            Types = new Type[] { typeof(int), typeof(float), typeof(int), noteTypeId.GetType(), typeof(bool) }, 
            Args = new object[] { bmsNote.Lane, bmsNote.Time, perfectSample, noteTypeId, isLeftBool }, 
            Desc = "5개 파라미터 (lane, time, sample, type, leftRight)" 
        },
    };
    
    // 일반 생성자 시도
    foreach (var sig in constructorSignatures)
    {
        if (noteCreateData != null) break;
        
        try
        {
            var constructor = noteCreateDataType.GetConstructor(CONSTRUCTOR_FLAGS, null, sig.Types, null);
                if (constructor != null)
                {
                    noteCreateData = constructor.Invoke(sig.Args);
                    if (EnableConstructorLogging)
                        MelonLogger.Msg($"[NoteConstructorHelper] {sig.Desc} 생성자 찾기 성공: {sig.Types.Length}개 파라미터");
                    break;
                }
        }
        catch (Exception ex)
        {
            ErrorLogger.LogWarning(ex, "[NoteConstructorHelper]", $"{sig.Desc} 생성자 호출 실패");
        }
    }
    
    // 6개 파라미터 생성자 시도 (Direction 추가)
    if (noteCreateData == null && bmsNote.Direction.HasValue)
    {
        noteCreateData = TryCreateWith6ParamsWithDirection(noteCreateDataType, bmsNote, perfectSample, noteTypeId, isLeftBool);
    }
    
    // 6개 파라미터 생성자 시도 (기본 Direction)
    if (noteCreateData == null)
    {
        noteCreateData = TryCreateWith6ParamsDefaultDirection(noteCreateDataType, bmsNote, perfectSample, noteTypeId, isLeftBool);
    }
    
    return noteCreateData;
}
```

**생성자 시도 순서:**

1. 2개 파라미터: `(float time, int lane)` 또는 `(int sample, int lane)`
2. 3개 파라미터: `(float time, int lane, NoteTypeId type)`
3. 4개 파라미터: `(float time, int lane, NoteTypeId type, bool leftRight)`
4. 5개 파라미터: `(int lane, float time, int sample, NoteTypeId type, bool leftRight)`
5. 6개 파라미터: `(int lane, float time, int sample, NoteTypeId type, bool leftRight, NoteDirectionIndex direction)`

## NoteFieldInitializer - 필드 초기화

### SetDirectionIndex 메서드

**목적**: directionIndex 필드 설정 (노트 타입에 따라 다름). **처리 순서**: FairyEnd → Direction 있음 → 홀드 레인 → 기본값.

```csharp
public static object SetDirectionIndex(object noteCreateData, BmsNote bmsNote)
{
    object directionIndexValue = null;
    
    if (bmsNote.Type == NoteType.FairyEnd)
    {
        // 페어리 끝(1A/1B): 위치는 곡선 깊이용 CENTER_TOP. 1A/1B는 turnDirection에서만 반영.
        directionIndexValue = EnumValueHelper.GetEnumValue(..., GetEnumCenterTop());
        if (directionIndexValue == null) directionIndexValue = GetEnumCenterMiddle();
    }
    else if (bmsNote.Direction.HasValue)
    {
        // 플릭/페어리 시작: BMS에서 지정한 방향 사용
        directionIndexValue = EnumValueHelper.GetDirectionIndex(bmsNote.Direction.Value);
    }
    else
    {
        if (bmsNote.Type == NoteType.Hold)
            directionIndexValue = EnumValueHelper.GetDirectionIndexFromLane(bmsNote.Lane, bmsNote.IsLeft);
        else
            directionIndexValue = EnumValueHelper.GetEnumValue(..., GetEnumCenterMiddle());
    }
    
    FieldAccessHelper.SetFieldValue(noteCreateData, FIELD_DIRECTION_INDEX, directionIndexValue);
    return directionIndexValue;
}
```

### SetTurnDirection 메서드

**목적**: turnDireciton 필드 설정 (페어리 노트용). **시그니처**: `SetTurnDirection(object noteCreateData, object directionIndexValue, BmsNote bmsNote = null)`.

**우선순위**:
1. **FairyEnd(1A/1B)**: `bmsNote.Direction` → Left = 1A, Right = 1B (레인 무관).
2. **Fairy 시작**: `bmsNote.EndNote.Direction`으로 턴 방향 결정 (끝 노트가 지시하는 방향).
3. **그 외**: `directionIndexValue` 문자열에서 "LEFT"/"RIGHT" 추출.

```csharp
public static void SetTurnDirection(object noteCreateData, object directionIndexValue, BmsNote bmsNote = null)
{
    string turnDirection = null;
    if (bmsNote != null && bmsNote.Type == NoteType.FairyEnd && bmsNote.Direction.HasValue)
    {
        turnDirection = bmsNote.Direction.Value == NoteDirection.Left ? GetEnumLeft() : GetEnumRight();
    }
    else if (bmsNote != null && bmsNote.Type == NoteType.Fairy && bmsNote.EndNote?.Direction.HasValue == true)
    {
        turnDirection = bmsNote.EndNote.Direction.Value == NoteDirection.Left ? GetEnumLeft() : GetEnumRight();
    }
    else if (directionIndexValue != null)
    {
        var directionStr = directionIndexValue.ToString();
        turnDirection = directionStr.IndexOf("LEFT", StringComparison.OrdinalIgnoreCase) >= 0 ? GetEnumLeft() : GetEnumRight();
    }

    if (turnDirection == null) return;
    // 필드 타입이 Enum이면 GetEnumValue(fieldType, turnDirection), string이면 그대로 SetFieldValue
    // fieldType이 Enum이면 GetEnumValue로 변환 후, SetFieldValue(noteCreateData, FIELD_TURN_DIRECTION, value)
}
