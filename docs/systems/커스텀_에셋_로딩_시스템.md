# 커스텀 에셋 로딩 시스템

## 개요

커스텀 이미지, 오디오 파일을 로딩하고 관리하는 시스템을 분석합니다.

## CustomAssetManager 클래스

### 구조 (현재 코드 기준)

```csharp
public static class CustomAssetManager
{
    private static Sprite customArtworkSprite = null;
    private static AudioClip customPreviewBGM = null;
    private static bool isCustomChartSelected = false;
    private static object customChartMusicID = null;
    private static string lastLoadedImagePath = null;  // 동일 경로 재로드 방지용
}
```

- **이미지 캐시**: `Dictionary`가 아닌 `lastLoadedImagePath`와 `customArtworkSprite`로 "같은 경로면 재로드 생략"만 수행합니다.
- **주입 제어**: `IsSceneWhereInjectionDisallowed()`(SoundPlayerScene, MoviePlayer_MovieSelect면 `true`), `ShouldInjectCustomContent()`(금지 씬 아님 + `IsCustomChartSelected()`)로 모든 주입 경로를 통제합니다.

### 커스텀 아트워크 로딩

```csharp
// 시그니처: 경로를 인자로 받음 (호출측에서 AlbumManager.GetCurrentImageFile() 등으로 전달)
public static void LoadCustomArtwork(string imagePath)
{
    if (!File.Exists(imagePath))
        return;

    // 같은 경로면 재로드 생략
    if (lastLoadedImagePath != null &&
        lastLoadedImagePath.Equals(imagePath, StringComparison.OrdinalIgnoreCase) &&
        customArtworkSprite != null)
    {
        return;
    }

    // 기존 스프라이트 해제 후, 파일에서 로드
    // 리사이즈/리샘플 없이 원본 해상도 그대로 사용
    // Sprite.Create(..., pixelsPerUnit: 51.2f), 이름 설정, lastLoadedImagePath = imagePath
}
```

- 반환값 없음 (`void`). 로드된 스프라이트는 `GetCustomArtwork()`로 조회합니다.
- `IsImageLoaded(string imagePath)`로 특정 경로가 이미 로드된 상태인지 확인할 수 있습니다.

### 커스텀 프리뷰 BGM 로딩

```csharp
public static AudioClip LoadCustomPreviewBGM()
{
    try
    {
        var bgmPath = AlbumManager.GetCurrentBgmFile();
        if (string.IsNullOrEmpty(bgmPath))
        {
            MelonLogger.Warning("[CustomAssetManager] BGM 파일이 없습니다.");
            return null;
        }

        if (!File.Exists(bgmPath))
        {
            MelonLogger.Error($"[CustomAssetManager] BGM 파일이 없습니다: {bgmPath}");
            return null;
        }

        // UnityWebRequest로 오디오 로드
        var audioType = GetAudioType(bgmPath);
        var uri = new Uri(bgmPath).AbsoluteUri;
        
        using (var www = UnityWebRequestMultimedia.GetAudioClip(uri, audioType))
        {
            www.SendWebRequest();
            
            while (!www.isDone)
            {
                // 대기
            }

            if (www.result != UnityWebRequest.Result.Success)
            {
                MelonLogger.Error($"[CustomAssetManager] BGM 로드 실패: {www.error}");
                return null;
            }

            var audioClip = DownloadHandlerAudioClip.GetContent(www);
            _customPreviewBGM = audioClip;

            MelonLogger.Msg($"[CustomAssetManager] BGM 로드 완료: {audioClip.length}초");
            return audioClip;
        }
    }
    catch (Exception ex)
    {
        ErrorLogger.LogException(ex, "[CustomAssetManager]", "BGM 로드 중 오류");
        return null;
    }
}
```

### 주입 제어 API (현재 코드 기준)

- **IsSceneWhereInjectionDisallowed()**: 현재 씬이 `SoundPlayerScene` 또는 `MoviePlayer_MovieSelect`이면 `true`. 이 씬에서는 BGM/BGA/노트 주입을 하지 않습니다.
- **ShouldInjectCustomContent()**: `IsSceneWhereInjectionDisallowed()`가 `false`이고 `IsCustomChartSelected()`가 `true`일 때만 `true`. 주입을 시도하는 모든 경로에서 사용합니다.
- **IsCustomChartSelected()** / **SetCustomChartSelected(bool)**: 커스텀 차트 선택 플래그.
- **IsCustomChart(object musicID, object musicData)**: MusicID·MusicData로 커스텀 차트 여부 판단.

## AssetLoader 클래스

### 이미지 로딩 유틸리티

```csharp
public static class AssetLoader
{
    public static Texture2D LoadTexture(string filePath)
    {
        try
        {
            if (!File.Exists(filePath))
            {
                MelonLogger.Error($"[AssetLoader] 파일이 없습니다: {filePath}");
                return null;
            }

            var imageData = File.ReadAllBytes(filePath);
            var texture = new Texture2D(2, 2);
            
            if (!texture.LoadImage(imageData))
            {
                MelonLogger.Error("[AssetLoader] 텍스처 로드 실패");
                return null;
            }

            return texture;
        }
        catch (Exception ex)
        {
            ErrorLogger.LogException(ex, "[AssetLoader]", "텍스처 로드 중 오류");
            return null;
        }
    }

    public static Sprite CreateSprite(Texture2D texture)
    {
        if (texture == null)
        {
            return null;
        }

        return Sprite.Create(
            texture,
            new Rect(0, 0, texture.width, texture.height),
            new Vector2(0.5f, 0.5f));
    }
}
```

## PlaySceneArtworkInjector

플레이 씬에서 커스텀 아트워크를 주입합니다. **주입 금지 씬**(`CustomAssetManager.IsSceneWhereInjectionDisallowed()`: SoundPlayerScene, MoviePlayer_MovieSelect)에서는 주입하지 않습니다. `GameObject.Find("ArtWork")`로 아트워크 Image를 찾아 단일 캐시(`_cachedArtWorkImage`)로 재사용하며, `ResetCache()`로 씬 재로드 시 캐시를 무효화합니다.

### 아트워크 주입

```csharp
public static class PlaySceneArtworkInjector
{
    private static UnityEngine.UI.Image _cachedArtWorkImage = null;
    private static int _lastSceneHash = 0;
    private static object _artworkInjectionCoroutine = null;

    public static void StartArtworkInjection()
    {
        ResetCache();
        if (TryInjectArtworkImmediately())
            return;
        _artworkInjectionCoroutine = MelonCoroutines.Start(InjectArtworkInPlaySceneCoroutine());
    }

    private static bool TryInjectArtworkImmediately()
    {
        try
        {
            if (CustomAssetManager.IsSceneWhereInjectionDisallowed())
                return false;

            var artwork = CustomAssetManager.GetCustomArtwork();
            if (artwork == null)
            {
                return false;
            }

            // GameObject.Find("ArtWork")로 Image 찾기, 캐시에 저장 후 sprite 설정
            var artWorkObj = GameObject.Find("ArtWork");
            if (artWorkObj != null)
            {
                var img = artWorkObj.GetComponent<UnityEngine.UI.Image>();
                if (img != null)
                {
                    _cachedArtWorkImage = img;
                    img.sprite = artwork;
                    return false;
                }
            }
            return false;
        }
        catch (Exception ex)
        {
            Helpers.ErrorLogger.LogWarning(ex, "[PlaySceneArtworkInjector]", "즉시 적용 중 오류");
            return false;
        }
    }

}
```

## 메모리 관리

### 이미지 재로드 방지 (CustomAssetManager)

현재 코드는 `Dictionary` 대신 **동일 경로 재로드 생략**만 수행합니다.

- `lastLoadedImagePath`와 `customArtworkSprite`가 있고, 요청 경로가 같으면 `LoadCustomArtwork(imagePath)`에서 재로드하지 않습니다.
- `IsImageLoaded(string imagePath)`로 해당 경로가 이미 로드된 상태인지 확인할 수 있습니다.

## 성능 최적화

### 1. 즉시 적용 시도

```csharp
// 코루틴 시작 전에 즉시 적용 시도
if (TryInjectArtworkImmediately())
{
    return; // 성공하면 코루틴 불필요
}

// 실패 시에만 코루틴 시작
MelonCoroutines.Start(InjectArtworkCoroutine());
```

### 2. Image 컴포넌트 캐싱 (PlaySceneArtworkInjector)

- `GameObject.Find("ArtWork")`로 찾은 Image를 `_cachedArtWorkImage`에 저장해 재사용합니다.
- 씬 해시(`_lastSceneHash`)로 씬이 바뀌면 캐시를 무효화하고, `ResetCache()`로 명시적 초기화가 가능합니다.

### 3. 원본 해상도 유지

```csharp
// ✅ 좋은 예: 원본 해상도 그대로 사용
var sprite = Sprite.Create(
    texture,
    new Rect(0, 0, texture.width, texture.height),
    new Vector2(0.5f, 0.5f));

// ❌ 나쁜 예: 리샘플링 (불필요한 성능 저하)
var resizedTexture = ResizeTexture(texture, 1024, 1024);
```

## 지원 파일 형식

### 이미지

- `.jpg`, `.jpeg`: JPEG
- `.png`: PNG
- 원본 해상도 유지
- 자동 스케일링 (Unity CanvasScaler)

### 오디오

- `.mp3`: MPEG
- `.wav`: WAV
- `.ogg`: OGG Vorbis

## 베스트 프랙티스

### 1. 파일 존재 확인

```csharp
// ✅ 좋은 예
if (!File.Exists(filePath))
{
    MelonLogger.Error($"파일이 없습니다: {filePath}");

```

### 2. 에러 처리

```csharp
// ✅ 좋은 예
try
{
    var texture = LoadTexture(filePath);
}
catch (Exception ex)
{
    ErrorLogger.LogException(ex, "[AssetLoader]", "로드 실패");
    return null;
}
```

### 3. 캐시 활용

```csharp
// ✅ 좋은 예: 캐시 확인 후 로드
if (_cache.TryGetValue(key, out var cached))
{
    return cached;
}

var loaded = Load(key);
_cache[key] = loaded;
return loaded;
```
