# 씬 관리 및 라이프사이클

## 개요

Unity 씬 전환과 모드의 라이프사이클 관리 시스템을 분석합니다.

## 씬 라이프사이클

```
게임 시작
    ↓
SceneDetector.OnInitializeMelon()
    ├─ MusicInjector.Initialize()
    ├─ AlbumManager.ScanAlbums(_hwaFolderPath)
    ├─ BMS 파일 파싱
    └─ Harmony 패치 적용
    ↓
씬 전환 감지
    ↓
SceneDetector.OnSceneWasLoaded(buildIndex, sceneName)
    ├─ MusicSelectScene → 로깅
    ├─ FairyModeScene / PlayMovieScene → HandleFairyModeScene/HandlePlayMovieScene, BGA/BGM 주입 시작, BgmGameEndMonitor 조정
    ├─ RenderCutinScene → 프리뷰 BGM 정리, BGA/BGM 주입, 커스텀 차트 시 아트워크 주입
    ├─ RythmGameResultScene → BgmBgaInjector 중지, ResultSceneInjector.StartInjection() (아트워크·난이도 주입)
    └─ SoundPlayerScene / MoviePlayer_MovieSelect → 주입 비활성화
```

## SceneDetector 클래스

### 초기화

```csharp
public override void OnInitializeMelon()
{
    try
    {
        MelonLogger.Msg("[GRC2] [Main] 모드 초기화 시작...");
        
        // 1. Steam 업데이트 차단
        SteamManifestLocker.LockManifest();
        
        // 2. 곡 목록 주입 시스템 초기화
        MusicInjector.Initialize();
        
        // 3. hwa 폴더 생성 (없으면 디렉터리 생성)
        var gameFolder = Path.GetDirectoryName(Application.dataPath);
        _hwaFolderPath = Path.Combine(gameFolder, "hwa");
        if (!Directory.Exists(_hwaFolderPath)) Directory.CreateDirectory(_hwaFolderPath);
        
        // 4. 앨범 스캔
        AlbumManager.ScanAlbums(_hwaFolderPath);
        
        // 5. BMS 변환기 초기화 (내부에서 GameTypeLoader.Initialize() 호출)
        BmsNoteConverter.Initialize();
        
        // 6. BMS 파일 스캔 및 파싱 (앨범별, ParsedBmsNotesByFile 캐시)
        ScanAndParseBmsFiles();
        
        // 7. 노트 배열 후킹
        NoteArrayHooks.Initialize(ParsedBmsNotes);
        
        // 8. BGM 주입 초기화
        BgmInjector.Initialize();
        BgmBgaInjector.Initialize(_hwaFolderPath);
        
        MelonLogger.Msg("[GRC2] [Main] 모드 초기화 완료!");
    }
    catch (Exception ex)
    {
        ErrorLogger.LogException(ex, "[SceneDetector]", "초기화 중 오류");
    }
}
```

### 씬 로드 처리

```csharp
public override void OnSceneWasLoaded(int buildIndex, string sceneName)
{
    // FairyModeScene: HandleFairyModeScene() → BgmBgaInjector.StartInjection(true), BgmGameEndMonitor.AdjustMusicDataOnSceneLoad() (구현: BgmGameEndMonitor.Adjustments.cs)
    // PlayMovieScene: HandlePlayMovieScene() → 동일
    // RenderCutinScene: CustomBgmPlayer.Cleanup(), BgmBgaInjector.StartInjection(true), 커스텀 차트 시 ReloadCurrentAlbumAssets(), PlaySceneArtworkInjector.StartArtworkInjection()
    // RythmGameResultScene: BgmBgaInjector.StopInjection(), ResetPlaySceneState(), ResultSceneInjector.StartInjection()
    // SoundPlayerScene / MoviePlayer_MovieSelect: StopInjection(), CustomAssetManager.SetCustomChartSelected(false)
    // MusicSelectScene: 로깅만. 기타 씬: 플레이 씬 상태면 아트워크 주입 시도, 아니면 BGA만 StartInjection(false)
}
    catch (Exception ex)
    {
        ErrorLogger.LogException(ex, "[SceneDetector]", $"씬 로드 처리 중 오류 ({sceneName})");
    }
}
```

## SceneHandler 클래스

### 플레이 씬 처리

```csharp
public static void HandlePlayScene(string sceneName)
{
    try
    {
        MelonLogger.Msg($"[SceneHandler] 플레이 씬 처리: {sceneName}");
        
        // 1. 프리뷰 BGM 중지
        StopPreviewBGMOnPlayScene();
        
        // 2. 현재 앨범 에셋 갱신
        ReloadCurrentAlbumAssets();
        
        // 3. BGA/BGM 주입 시작
        BgmBgaInjector.StartInjection(isPlayScene: true);
        
        // 4. 게임 종료 시간 조정
        BgmGameEndMonitor.AdjustMusicDataOnSceneLoad() (구현: BgmGameEndMonitor.Adjustments.cs);
        
        // 5. 플레이 씬 아트워크 주입
        PlaySceneArtworkInjector.StartArtworkInjection();
    }
    catch (Exception ex)
    {
        ErrorLogger.LogException(ex, "[SceneHandler]", "플레이 씬 처리 중 오류");
    }
}
```

## 씬 플래그 및 주입 제어

### 주입 금지 씬 (CustomAssetManager.IsSceneWhereInjectionDisallowed)

다음 씬에서는 **커스텀 BGM/BGA/노트 주입을 절대 수행하지 않습니다** (실행 순서와 무관하게 차단).

- **SoundPlayerScene**: 사운드 플레이어 씬
- **MoviePlayer_MovieSelect**: 무비 플레이어/영상 선택 씬

`CustomAssetManager.ShouldInjectCustomContent()`는 `IsSceneWhereInjectionDisallowed()`가 `true`이면 항상 `false`를 반환하며, `IsCustomChartSelected()`가 `true`이고 주입 금지 씬이 아닐 때만 주입을 허용합니다.  
NoteArrayHooks, BgmBgaInjector, GameFlowHooks, PlaySceneArtworkInjector, MusicScrollViewHooks, AudioClipPatch 등 **주입을 시도하는 모든 경로**에서 이 조건을 사용합니다.

## 씬별 동작

### MusicSelectScene (곡 선택 화면)

**초기화:**
- 커스텀 차트 선택 상태 초기화 (`SetCustomChartSelected(false)`)
- 텍스트 교체 비활성화

**주입 판단:** 플레이/주입 경로에서는 `CustomAssetManager.ShouldInjectCustomContent()`(선택 플래그 + 주입 금지 씬 아님)로 주입 여부를 판단합니다.

**Harmony 후킹:**
- `initializeAllItemByCrrentMusicData`: 곡 목록 주입
- `noticeChangedMusic`: 곡 선택 변경 감지
- `coOpen`: 곡 선택 UI 열기
- `coOpenPreMusicStartWindow`: 곡 시작 전 윈도우

### FairyModeScene / PlayMovieScene (플레이 화면)

**초기화:**
- 프리뷰 BGM 중지
- 현재 앨범 에셋 갱신
- BGA/BGM 주입 시작
- 게임 종료 시간 조정
- 아트워크 주입

**Harmony 후킹:**
- `createAllNote`: 노트 배열 주입
- `setClip`: BGM 주입
- `VideoPlayer.Play`: BGA 재생

### RythmGameResultScene (결과 화면)

**초기화:**
- BGM 주입 중지

## 에셋 갱신

### ReloadCurrentAlbumAssets

```csharp
private static void ReloadCurrentAlbumAssets()
{
    try
    {
        // 현재 앨범의 BMS 노트 캐시 갱신
        var currentBmsFile = AlbumManager.GetCurrentBmsFile();
        if (!string.IsNullOrEmpty(currentBmsFile))
        {
            var bmsNotes = BmsParser.ParseBmsFile(currentBmsFile, printSummary: false);
            SceneDetector.ParsedBmsNotes = bmsNotes;
        }
    }
    catch (Exception ex)
    {
        ErrorLogger.LogException(ex, "[SceneHandler]", "에셋 갱신 중 오류");
    }
}
```

## PlaySceneArtworkInjector

플레이 씬에서 커스텀 아트워크를 주입합니다. **주입 금지 씬**(`CustomAssetManager.IsSceneWhereInjectionDisallowed()`: SoundPlayerScene, MoviePlayer_MovieSelect)에서는 주입하지 않으며, `GameObject.Find("ArtWork")`로 아트워크 Image를 찾아 캐시(`_cachedArtWorkImage`)합니다. `ResetCache()`로 씬 재로드 시 캐시를 무효화합니다.

### 아트워크 주입 코루틴

```csharp
public static void StartArtworkInjection()
{
    ResetCache();  // 재시작 시 캐시 무효화
    
    // 즉시 적용 시도 (성능 최적화)
    if (TryInjectArtworkImmediately())
    {
        return;
    }
    
    // 실패 시 코루틴으로 재시도 (InjectArtworkInPlaySceneCoroutine)
    _artworkInjectionCoroutine = MelonCoroutines.Start(InjectArtworkInPlaySceneCoroutine());
}

private static IEnumerator InjectArtworkInPlaySceneCoroutine()
{
    yield return new WaitForSeconds(0.05f);
    
    if (CustomAssetManager.IsSceneWhereInjectionDisallowed())
        yield break;
    
    // ArtWork Image 찾기 및 스프라이트 적용 (캐시 활용)
    // 최대 재시도 횟수 내에서 주기적으로 적용 시도
}
```
